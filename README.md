# Senthuron Cafe Website

A fully responsive cafe website built with HTML, CSS, and JavaScript.

## Features
- **Hero Section** with call-to-action button
- **Menu Section** with 6 item cards (image, name, description, price)
- **Contact Section** with address, working hours, and functional contact form
- **Mobile Responsive** design that works on all devices
- **Interactive Elements** including mobile navigation and form validation

## Live Demo

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts

## How to Run Locally
1. Clone this repository
2. Open `cafe.html` in your browser
3. That's it! No build process required.
